# Security

Please read the [Security Policy][security policy] for this repository.

## Security Vulnerabilities

Security issues are treated very seriously and will, by default,
takes precedence over other considerations including usability, performance,
etc...  Best effort will be used to mitigate side effects of a security
change.

### Reporting a security vulnerability

Please refer to the [Security Policy][security policy]

[Security Policy]: <https://github.com/microsoft/containers-toolkit/security/policy>)
